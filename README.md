### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8或jdk17`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

学生功能：1. 登录、2. 注册、3. 首页、4. 服饰、5. 课程、6. 视频、7. 舞蹈交流、8. 论坛交流、9. 舞蹈资讯、10. 个人中心、11. 后台管理、12. 购物车

管理员功能：1. 登录、2. 个人中心、3. 会员用户管理、4. 服饰管理、5. 课程管理、6. 视频管理、7. 舞蹈交流管理、8. 服饰类型管理、9. 视频分类管理、10. 课程类型管理、11. 在线招生管理、12. 系统管理、13. 订单管理
